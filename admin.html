Instructions

Important: After making changes, you need to manually update the HTML files:

  1. Toggle room availability switches below
  2. Note which rooms changed status
  3. Open the corresponding property HTML file (property-1.html or property-2.html)
  4. Find the room card and change class="room-card available" to class="room-card taken" (or vice versa)
  5. Change the room-status div text from "Available" to "Taken" (or vice versa)
  6. Save and push changes to GitHub

This dashboard helps you track changes. For a fully automated system, you'd need a backend database.
